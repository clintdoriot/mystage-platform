# MyStage FlutterFlow Base Library

## Overview

Base FlutterFlow project that serves as the foundation for other FlutterFlow applications.

## Purpose

Provides shared components and functionality for:
- **mystage-app** - Mobile application
- **mystage-ff-fanex** - Live performance interaction site
