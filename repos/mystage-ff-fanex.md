# MyStage Fanex

## Overview

FlutterFlow-based website for live performance interaction between fans and performers.

## Technology Stack

- **Platform**: FlutterFlow
- **Deployment**: Web
- **Base**: Extends mystage-ff-base-lib

## Purpose

Live performance features including:
- **Tipping** - Digital tipping for performers
- **Song Requests** - Fans can request songs
- **Chat** - Real-time chat during performances

## Development Process

- Built using FlutterFlow visual development platform
- Git is used as **backup only**, not primary development workflow
- Changes typically made through FlutterFlow interface

## Related Repositories

- **mystage-ff-base-lib** - Base FlutterFlow project this extends
- **mystage-app-backend** - Backend functions for tipping/chat/requests
- **mystage-databases** - Database for live interaction data

## Current Status

Active development. Needs better documentation and Claude tooling setup.

## Notes

- FlutterFlow-based means limited traditional git workflow
- Real-time features require careful backend design
- May have unique payment processing requirements for tipping
